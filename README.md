# Task Management App

A simple task management application built using HTML, CSS, and JavaScript.

## Description

This project allows users to add, edit, and delete tasks directly in the browser. It is designed to practice DOM manipulation, event handling, and basic frontend development.

## Features

- Add new tasks
- Edit existing tasks
- Delete completed or unwanted tasks
- Clean and responsive user interface

## Files and Their Purpose

- `index.html` - Creates the structure of the task management application.
- `style.css` - Adds styling, layout, colors, and responsive design.
- `script.js` - Handles task adding, editing, deleting, and DOM manipulation.
- `README.md` - Explains the project, features, and file purposes.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in any web browser.
